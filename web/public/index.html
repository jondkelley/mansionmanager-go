Media proxy & TLS edge

Mirrors /etc/palace-manager/config.json (nginx.*). The computed URL is passed to each pserver as --reverseproxymedia. Changing the edge scheme rewrites every palman-*.service unit — restart palaces so running processes publish the new URL.

When set, new palaces are provisioned with --provider and this name. Leave empty to omit that flag.

Backups

Each day at midnight UTC, the manager copies pserver.pat, pserver.prefs, and serverprefs.json (when present) into backups/ under the palace data directory, using names like pserver.pat-mm-dd-yy.bak. After each run, older snapshots are removed so only the 30 most recent dated files per config remain. Take backup now writes the same snapshots on demand (every file that exists is copied into backups/ for today’s UTC date tag, overwriting today’s files if you run it again). Download backup is a full gzip tarball of the palace Unix home (the on-disk backups/ tree is excluded from that archive).
